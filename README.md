# Sivert Programming Language
A simple programming language written in C++ for study purpose

## Build using CMake
### Linux and macOS

```sh
git clone https://github.com/LuizSudo/Sivert
cd Sivert
mkdir build && cd build
cmake -B build .
cmake --build build

```
### Windows
> [!NOTE]
> Microsoft Visual C++ Compiler is required

> Open the powershell
```powershell
git clone https://github.com/LuizSudo/Sivert
cd Sivert
mkdir build && cd build
cmake -B build .
cmake --build build
```
